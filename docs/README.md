## üöÄ Quick Start

1. **Install Merobox**: `pip install merobox`
2. **Start nodes**: `merobox run --count 2`
3. **Create workflow**: `merobox bootstrap create-sample`
4. **Execute workflow**: `merobox bootstrap run workflow-sample.yml`
